Zenbox API and SDK

Zenbox provides an API for making voice requests and fetch results from your Zenbox projects through your application on any device. This allows you to build a voice enabled apps and devices.

Zenbox SDKs simplify the work with this API and also provide a ready-for-use speech recognition and speech synthesis components. Thus you don't need to implement everything on your own.

Select your platform

To start using Zenbox you have to select an appropriate SDK for your device's platform or use an HTTP API.
